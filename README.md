# card_game

## Deployment Step
1. Install Docker and start Docker
2. Clone this github repository (https://github.com/chotikalu/card_game.git)
3. Go to /card_game
4. Run docker-compose build
5. Run docker-compose up
6. Access api using http://0.0.0.0:8000/docs
